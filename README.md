# 📊 Data Analysis & Visualization Project

## 📌 Project Overview

The **Data Analysis & Visualization Program** is a Python-based interactive project designed to load, explore, analyze, and visualize datasets using powerful libraries like **Pandas** and **Matplotlib**.

This project provides a menu-driven interface that allows users to perform various data operations and generate visual insights from CSV datasets.

---

## 🎯 Project Objectives

* Load and analyze CSV datasets efficiently
* Explore dataset structure and content
* Perform sorting and filtering operations
* Handle missing data effectively
* Generate descriptive statistics
* Create different types of visualizations
* Save generated plots as image files

---

## ⚙️ Key Features

### 1. 📂 Load Dataset

Users can load any CSV file by providing the correct file path.

* Supports external dataset loading
* Error handling for invalid paths
* Displays confirmation after successful load

---

### 2. 🔍 Explore Data

This module helps users understand the dataset:

* View first 5 rows (`head`)
* View last 5 rows (`tail`)
* Display column names
* Check data types
* Get dataset information (`info`)

---

### 3. 🛠️ DataFrame Operations

Users can manipulate data easily:

* Sort data based on any column
* Filter data using specific values

This helps in organizing and extracting meaningful data.

---

### 4. ⚠️ Handle Missing Data

This feature includes:

* Display rows with missing values
* Fill missing values using mean
* Drop rows with missing values
* Replace missing values with custom input

---

### 5. 📈 Descriptive Statistics

Users can generate statistical summaries:

* Count
* Mean
* Standard Deviation
* Minimum & Maximum values
* Quartiles

Using `df.describe()` for quick insights.

---

### 6. 📊 Data Visualization

The program supports multiple visualization types:

* 📊 Bar Plot
* 📈 Line Plot
* 🔵 Scatter Plot
* 🥧 Pie Chart
* 📉 Histogram

These visualizations help in better understanding data trends and patterns.

---

### 7. 💾 Save Visualization

Users can save generated plots:

* Save plots as `.png` files
* Useful for reports and presentations

---

## 🗂️ Data Handling

* Works with CSV files
* Uses Pandas DataFrame for processing
* Handles both numerical and categorical data

## 🚀 How to Run

1. Install required libraries:

   ```bash
   pip install pandas matplotlib
   ```

2. Run the Python file:

   ```bash
   python your_file_name.py
   ```

3. Enter the CSV file path when prompted:

   ```
   C:/Users/YourName/Downloads/file.csv
   ```

---

## ⚠️ Important Notes

* Always provide full file path including `.csv` extension
* Ensure dataset is available at the given location
* Use `/` instead of `\` in file paths
